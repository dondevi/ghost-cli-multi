# ghost-dev-multiple
> for ghost.js 1.22.1


## Initialation
> <https://docs.ghost.org/v1.0.0/docs/install-local>{target="\_blank"}

```bash
  # If use NVM
  $ npm run init:nvm

  # If not install ghost-cli
  $ npm run init:cli

  # If not install ghost
  $ npm run init
```

## Development
> <https://themes.ghost.org/>{target="\_blank"}

### Add Project

```bash
  $ npm run add
  # Then input project name
```

### Develop Project

```bash
  $ cd ${project_name}
  $ npm run dev
```
